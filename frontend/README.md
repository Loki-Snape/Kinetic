# Kinetic Frontend 🎨

This directory contains the client-side SPA code for the **Kinetic** Kanban board.

## 🛠️ Stack & Libraries
*   **React** (v18+) & **Vite**
*   **Tailwind CSS** (for styling & glassmorphic utility layers)
*   **@hello-pangea/dnd** (fluid drag & drop interactions)
*   **lucide-react** (modern UI icons)
*   **axios** (HTTP requests with auto-logout/401 interceptors)
*   **socket.io-client** (real-time live board updates)
*   **react-router-dom** (protected routes & authentication flow)

## 🚀 Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the Vite development server:
   ```bash
   npm run dev
   ```

3. Open **[http://localhost:5173](http://localhost:5173)** in your browser.
